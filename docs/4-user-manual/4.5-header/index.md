# 4.5 Header

The header is always visible at the top of the application and
provides quick access to global controls.

## Application icon and title

The leftmost area of the header shows the application icon and title.
Clicking either element returns to the dashboard from any page.

## Theme toggle

The theme toggle cycles between three display modes:

- **System preference** — the application follows the operating system
  dark or light mode setting.
- **Light mode** — forces a light colour scheme regardless of the
  system setting.
- **Dark mode** — forces a dark colour scheme regardless of the system
  setting.

Click the icon to advance to the next mode. The selected mode is
persisted across sessions.

## Seeded favourite toggle

The seeded favourite toggle shows or hides the default `tiogars/template-repository`
template on the dashboard without permanently deleting it. This is
useful when you want to focus on your own templates.

- When the toggle is **on**, the seeded template is visible on the
  dashboard.
- When the toggle is **off**, the seeded template is hidden but
  retained in the local database.

The toggle state is persisted across sessions.

## Documentation link

The documentation icon in the header opens the project documentation
in a new tab.
